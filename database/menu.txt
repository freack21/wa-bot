*>BOT MENU<*
*-menu* or *-help* -> view all menu
*-bot* -> summon bot

*>BASIC FEATURES<*
*-groupinfo* -> see group info
*-everyone* or *-tagall* -> tag everyone in group
*-hidetag* -> send a message and tag everyone

*>STICKERS & MEDIA<*
*-sticker* or *-s* -> make sticker
*-stickers* -> view available random stickers
*-smeme* -> make meme sticker
*-meme* -> make meme from image
*-emojis* -> view available emoji to sticker platform
*-toimg* or *-img* -> convert sticker to image

*>DOWNLOADER<*
*-yt* or *-ytvid* 'url' -> download video from YouTube
*-ytaudio* 'url' -> download audio from YouTube
*-tt* or *-tiktok* 'url' -> download video and audio from TikTok
*-ig* 'url' -> download video from Instagram

*>PRAY<*
*-solat* or *-salat* 'city' -> view Solat Schedule
*-doa* 'all' -> view all available doa
*-doa* 'name' -> view doa

*>PHONE SPECIFICATIONS<*
*-speq* 'keyword' -> search phones name by keyword
*-spec* 'phone name' -> check specifications phone by query
*-spek* 'keyword' -> check specifications phone by keyword

*>AI<*
*-ai* or *-t* -> talking with bot
*-tlen* -> translate to english
*-tlin* or *-tlid* -> translate to bahasa indonesia
*-stden* -> correct sentences to Grammatical Standard English

*>GAMES<*
*-gamecodes* -> view available gamecode
*-fam* -> start Family100
*-lontong* -> start Quiz Lontong
*-caklontong* -> start Cak Lontong
*-tebakkata* -> start Tebak Kata
*-tebaklirik* -> start Tebak Lirik
*-tebakkalimat* -> start Tebak Kalimat
*-qdd* -> add question for selected gamecode
*-stop* 'gamecode' -> stop game from selected gamecode
*-lb* -> view leaderboard in this group

*>NEWS<*
*-cnn* 'category' -> search news by category in CNN indonesia
*-category* 'newsportal' -> view available categories of newsportal
